Windows配置Git

2019-08-26 软件

安装

​ 官方版本可以在 Git 官方网站下载。 打开 http://git-scm.com/download/win，下载会自动开始。要注意这是一个名为 Git for Windows 的项目（也叫做 msysGit），和 Git 是分别独立的项目；更多信息请访问 http://msysgit.github.io/。安装教程：https://blog.csdn.net/qq_32786873/article/details/80570783 安装完成后，下一步。

注册GitHub

​ 托管代码的地方。比较注明的有GItHub、码云。这里使用GitHub，他们没有什么区别，码云是中国的，GItHub不是中国的。访问速度码云比较快。

​ 打开网址：https://github.com/join?source=header-home按照提示完成注册，并登录。需要一定的英文基础。注册教程：https://blog.csdn.net/JIANGXIN04211/article/details/89187801 注册完成后，下一步

创建GitHub仓库

登录注册的GitHub账号，来到以下界面（如有更新，界面可能略有不同）

点击New repository，来到如下界面。

填写相应信息，完成注册。

附上常用命令：

git pull origin master 加载服务器上的文件，和本地保持一致。

git add . 添加到上传列表。

git config –global user.user “xx” 配置全局的用户名

git config –global user.user 查看全局的用户名

git config –gloval user.email “xx” 配置全局的邮箱

git config –gloval user.email 查看全局的邮箱

git commit -m “xx” 上传的备注

git push origin master 正式上传文件到主分支
