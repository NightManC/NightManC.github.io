SpringMVC-json

2019-08-26 前端

Json

​ 简介：Object，拥有属性和方法。

​ 课外：所有编程语言都支持json和xml，先有xml过度到json。

​ 一个对象：{ key:”value”}

​ 一个数组：[{ key:”value”},{ key:”value”}]

​ java中，json中的key必须加双引号；Javascript中key是可以不用双引号的，而且绝大部分人不写。

​ Handl ：负责人、处理器

第三方json插件

​ jackson是SpringMVC默认使用的

  • jackson-databind

  • jackson-core

  • jackson-annotations

    FastJSON是阿里云发行的，性能比jackson要好很多

操作

  1. 在上一篇博客Spring的基础上。

  2. pom.xml文件内添加jackson的三个引用。

  3. 创建实体类

  4. 创建Test类

  5. springmvc.xml文件内添加

  6. 启动Tomcat，进入localhost:8080/jsp，页面将输出json数据
